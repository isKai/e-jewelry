# e-jewelry
基于区块链技术之可溯源珠宝电商平台



**使用环境**

1. Truffle v5.1.45 (core: 5.1.45)
2. Solidity v0.5.16 (solc-js)
3. Node v12.14.1
4. Web3.js v1.2.1

**快速上手**

```shell
# 将源码下载到本地
git clone https://github.com/isKai/e-jewelry.git
# 进入项目目录
cd e-jewelry
# 编译
truffle compile
# 部署(打开ganache)
truffle migrate
# 运行前端
npm run dev
```
